# 🤖 FAQ Chatbot 

A simple yet effective **FAQ Chatbot** built using **Python**, **Natural Language Processing (NLP)**, and **Machine Learning techniques**. The chatbot analyzes user questions, finds the most similar question from a predefined FAQ dataset using **TF-IDF Vectorization** and **Cosine Similarity**, and returns the best matching answer.

---

## 📌 Features

- 📚 FAQ dataset stored in CSV format
- 📝 Text preprocessing using NLTK
- 🔤 Tokenization
- 🔡 Lowercase conversion
- 🚫 Stopword removal
- 🧹 Punctuation removal
- 📊 TF-IDF Vectorization
- 📐 Cosine Similarity for question matching
- 💬 Interactive command-line chatbot
- ⚡ Fast and lightweight implementation

---

## 🛠️ Technologies Used

- Python 3
- Pandas
- NumPy
- NLTK
- Scikit-learn

## 🔄 Workflow

```

      │

      ▼
```
